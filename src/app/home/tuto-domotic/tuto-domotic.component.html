Tutorial to create a domotic project:

This tutorial will show you:
  • How to create a new user account
  • How to create a new project
  • How to handle the tree-view
  • How to handle the tab-view
How to create a new user account
The first time you connect to the web, you have to create an user account, to do this please go to the top, and on the right side you have to click the button named "SignIn", this action will make appear a pop-up named "Authentification". As you don't have an user account you have to click on the button named "SignUp", this action will make appear a pop-up named "SignUp" also, to make an account you have to fill out the form. Then you can click on "SignIn" to connect.
How to create a new project
To create a new project yo have to connect first; then a new page will appear, and the right you can add a new project by clicking on the button IMAGE HERE , the button IMAGE HERE help you to edit and to modify you project's informations, the button IMAGE HERE will delete your project. you can have many projects as you want. To create your domotic specifications you have to click on the project you create first, then a new page will appear, this page contains on the right side a tree-view and on the left side a tab-view.
